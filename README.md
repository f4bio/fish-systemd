# systemd plugin for oh-my-fish

based on https://github.com/robbyrussell/oh-my-zsh/blob/master/plugins/systemd/systemd.plugin.zsh

## implemented

### sudo
* scdisable
* scenable
* screload
* screstart
* scstart
* scstatus
* scstop

### user
* scudisable
* scuenable
* scureload
* scurestart
* scustart
* scustatus
* scustop
